Multi-Modal Fusion: Integrated stereo vision with IMU for quadrotor pose estimation and depth perception using ESKF framework

3. ESTIMATE

I implement probabilistic filters that maintain real-time belief states under measurement noise and model uncertainty.

Proof Points:

Visual-Inertial Odometry: Implemented Error State Kalman Filter (ESKF) for VIO on CrazyFlie 2.0 quadrotor

Embedded Filtering: Developed custom filtering circuits and localization algorithms for autonomous battlebot (ESP32 bare-metal)

Dynamic Tracking: Integrated perception and state estimation for dynamic object tracking in robotic manipulation tasks

Artifact: Side-by-side comparison showing raw noisy sensor data vs. ESKF-filtered smooth state estimate with covariance ellipses

4. MODEL

I learn data-driven models of nonlinear dynamics that enable predictive control and anomaly detection.

Proof Points:

Physics-Informed Learning: Explored PINN and DeepONet to learn nonlinear dynamics for Robust/Hybrid/Explicit MPC at GRASP Lab

Health Monitoring: Implemented real-time robot health monitoring using Autoregressive models and Neural ODEs at Verdant Robotics

Embedded ML: Deployed TinyML and non-convex optimization solvers on resource-constrained Cortex-M microcontrollers

Artifact: Neural ODE learning robot dynamics ‚Üí Predicted vs. Actual trajectory comparison ‚Üí Anomaly detection heatmap

I design model-based controllers that stabilize underactuated systems and track aggressive trajectories with provable guarantees.

Proof Points:

Nonlinear Control: Designed geometric nonlinear control for quadrotor with iLQR/MPC using Drake and CaSaDi frameworks

Motor Control: Developed Field-Oriented Control (FOC) algorithms for high-speed BLDC motors on moteus firmware using Simulink Coder

Production Systems: Implemented thermal control algorithms and PID loops for precision temperature regulation in consumer products ($550M+ combined revenue)
